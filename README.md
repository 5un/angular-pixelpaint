# Angular Pixelpaint

Pixel paint directive for angular with multilayer, text rendering support.
Requires Hammerjs to be installed.

## Features
- Multilayer
- Brush Color
- Directive

#### Using the directive
```shell
npm install pixel-paint
```
```shell
bower install hammerjs
bower install pixel-paint
```
